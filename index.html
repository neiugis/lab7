Prepare Time Series Data

The first step for the spatiotemporal visualization is to create a time series dataset. For this lab, we'll map population dynamics of fifteen major U.S. cities since 1950 at 10-year intervals (You are free to use other spatiotemporal datasets if you prefer). Below is an example of what the data should look like (a .csv file):

  • Click here to download the csv file. Spend some time to make sure you understand the data structure (you will need to modify the data for answering Question 2).
  • In your web browser, go to geojson.io, select Open - File (see image below) to open the csv file you have just created.

  • The data, i.e., 15 cities, should have been loaded as points/markers.
  • Next, click Save - GeoJSON to save the data as a geojson file.

  • The last step is to CHANGE THE FILE EXTENSION of map.geojson from .geojson to .json (The file, i.e., something like map.geojson, should be found in the Downloads folder). If you are not sure, you may open the geojson file in Atom and then choose Save As to save the file as map.json. Then, move the map.json file to your lab folder.
